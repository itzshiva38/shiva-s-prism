

# Shiva's Ultra-Premium Glassmorphic Link-in-Bio Page

## Overview
A stunning single-page link-in-bio landing page with a dark theme, vibrant animated backgrounds, and consistent glassmorphic design system throughout.

## Design System
- Reusable `GlassCard` component enforcing exact glassmorphic specs (blur, borders, inner glow, hover states)
- Dark vibrant background with animated gradient mesh orbs in purple, magenta, cyan, electric blue
- CSS-based animated floating orbs/particles for the background
- All text with subtle text-shadow for depth

## Page Sections

### 1. Hero Section (full viewport on mobile)
- Animated gradient mesh background covering the entire page
- Centered glassmorphic hero card with:
  - Circular avatar with glowing ring border
  - "Shiva" name with verified badge
  - Tagline & short bio
  - Location tag: "Kota, Rajasthan, India 🇮🇳"

### 2. Links Section
- 9 vertical glassmorphic link buttons (YouTube, Instagram, X, Portfolio, Spotify, Discord, Newsletter, Buy Me a Coffee, Contact)
- Each with left icon, title, description, animated right arrow on hover
- Staggered fade-in animations on scroll

### 3. Bonus Sections
- "Now Playing" glassmorphic mini music player with fake album art, track info, progress bar, play/pause
- Horizontal scrollable "Featured" row with 3 small glass cards (Latest Video, Featured Project, Recent Post)

### 4. Footer
- Glassmorphic footer with social icon row (Instagram, X, YouTube, LinkedIn)
- "© 2026 Shiva • Made with ❤️ in India"

## Technical Approach
- Install `framer-motion` for smooth animations
- Custom CSS animated background with floating gradient orbs
- `GlassCard` reusable component with all exact glassmorphic values
- Mobile-first responsive layout
- Update page title to "Shiva | Link in Bio"
